# nextjs-geist-font-example

A modern **Next.js** starter template featuring [Geist](https://vercel.com/font) font integration — Vercel's clean, developer-friendly typeface designed for code and UI.

## Tech Stack

- **Framework**: Next.js 14+
- **Font**: Geist (Sans & Mono) by Vercel
- **Language**: JavaScript / React
- **Styling**: CSS Modules / Tailwind CSS

## Features

- Pre-configured Geist font (Sans & Mono variants)
- Optimized font loading via `next/font`
- Clean project structure ready to build on
- Fast development server with hot reload

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## About

This repo serves as a reference/starter for Next.js projects using Vercel's Geist font. Part of my frontend experimentation and learning projects.
